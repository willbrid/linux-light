# Linux Light

Ce référentiel est une ressource complète pour les passionnés de Linux, couvrant une multitude de sujets et de compétences clés nécessaires pour maîtriser les systèmes Linux, incluant **Red Hat** (>= 8), **Rocky Linux** (>= 8) et **Ubuntu** (>= 20.04).

### Contenu

- Scripts utiles
- Guides de configuration système
- Préparation aux certifications Linux
